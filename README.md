# PEroxide

A file scanning application built with React, TypeScript, and Rust.

> **⚠️ WIP Hobby Project**: This is a personal project and not production-ready software.

## Overview

PEroxide is a file scanner that analyzes files for potential malware. It features a clean UI with real-time scanning progress and detailed threat reporting.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Backend**: Rust (Actix/Tiny-Http)
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: Zustand + TanStack Query

## Quick Start

### Prerequisites
- Node.js 18+
- Rust 1.70+ (for backend)


### Usage
1. Open http://localhost:5173
2. Drag & drop files to scan
3. View real-time progress and results
